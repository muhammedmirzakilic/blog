# hexo-theme-Tsu
> A light and simple theme bases on polarbear, designed by Bacbin

## Installtion
```
$npm install hexo-rederer-scss --save
$git clone https://github.com/zjx137/hexo-theme-Tsu.git
```
Change config.yml`theme: Tsu` or`theme: hexo-theme-Tsu`
